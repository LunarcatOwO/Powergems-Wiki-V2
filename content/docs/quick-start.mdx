<Callout title="Info">
Before you begin, make sure your server supports plugins. We recommend [PaperMC](https://papermc.io/downloads/paper).
</Callout>
<div className='fd-steps [&_h3]:fd-step'>

### Download the plugin
Download it from the only from the official source which is [Modrinth](https://modrinth.com/plugin/powergems)

### Download the dependency plugin
Download it from the only from the official source which is [Modrinth](https://modrinth.com/plugin/seallib)

### Installing the plugin
Open your server panel, navigate to the file manager, go to `~/plugins/`, upload the plugin there, and **restart** the server.

### Configure the Plugin
Configuration can be challenging sometimes. For detailed instructions and an explanation of each option go to the [configuration section](/fundamentals/getting-set-up/configuration). If you are having trouble, feel free to contact us on [Discord](https://discord.iseal.dev); we will be happy to help!
</div>
